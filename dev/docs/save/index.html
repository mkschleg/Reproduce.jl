Reproduce.get_sql_name — Method
get_sql_name(name, param)

Returns the name used for a column. For a single value (i.e. String, Float, Integer, etc...) return name For a Tuple or a Vector return (name1, name2, ..., namen) For a NamedTuple return (name(prop1), name_(prop2), ...) where the props are sorted.
